Shopify via MCP

Claude connects directly to Shopify's product catalogue — searching inventory, comparing prices across sellers, and returning live checkout links — all through a single conversation.

Try the Live Demo → How it works
What it is

Shopify, connected to Claude

Model Context Protocol (MCP) is an open standard that lets AI models like Claude call external tools during a conversation. When Shopify is connected as an MCP server, Claude can search its product index, retrieve pricing from multiple sellers, and hand you a direct checkout URL — without you ever leaving the chat.

This isn't a chatbot that guesses at product details. Claude calls the live Shopify API on your behalf and works with real data: actual product titles, current prices, seller ratings, and working checkout links.

Use Cases

What you can do with it

Natural Language Product Search

Ask "find me a standing desk under $400" and Claude searches the catalogue, filters by price, and returns the best matches with descriptions.

Price Comparison

For any product, Claude fetches all available seller offers side-by-side — price, shipping cost, seller rating — so you always get the best deal.

Instant Checkout Links

Claude returns a direct, ready-to-use checkout URL for the product and seller of your choice. One click takes you straight to the cart.

Product Research

Get full product details — specs, images, descriptions, category — to make informed decisions or populate a database without manual browsing.

Store Integrations

Developers can embed Claude + Shopify MCP into their own apps — build a shopping assistant, a price-alert bot, or an inventory Q&A tool in minutes.

How it works

Three steps, zero friction

01

You send a message

Type a plain-English request: "Find wireless headphones under $100."

02

Claude calls Shopify via MCP

Claude invokes the MCP search tool, which queries the live Shopify product index in real time.

03

Results come back to you

Claude summarises the results, compares options, and hands you a direct checkout URL — all in one reply.

Available Tools

MCP tools Claude can use

productsearch_search_products Search millions of Shopify products by keyword, category, price range, or any natural language query.
productsearch_get_product_details Retrieve full product info — title, description, images, specs, and category — by Universal Product ID (UPID).
productsearch_compare_offers Compare all seller offers for a product side-by-side, sorted by total price including shipping.
productsearch_get_checkout_url Generate a direct checkout URL for a specific product from a specific seller — ready to click.
Code Example

Connecting Shopify via MCP

To use this in your own application, pass the MCP server URL in your Anthropic API call. Claude automatically discovers the available tools and calls them as needed.

const response = await fetch("https://api.anthropic.com/v1/messages", {
  method: "POST",
  headers: {
    "Content-Type": "application/json",
    "x-api-key": YOUR_ANTHROPIC_API_KEY,
    "anthropic-version": "2023-06-01",
    "anthropic-beta": "mcp-client-1-0"
  },
  body: JSON.stringify({
    model: "claude-sonnet-4-20250514",
    max_tokens: 1024,
    messages: [{
      role: "user",
      content: "Find me wireless headphones under $100"
    }],
    mcp_servers: [{
      type: "url",
      url: "https://api.mcpcio.com/mcp",  // Shopify MCP server
      name: "shopify-mcp"
    }]
  })
});

const data = await response.json();
console.log(data.content[0].text); // Claude's response with product results

Claude will automatically call productsearch_search_products and weave the results into a natural reply — no extra parsing required.
