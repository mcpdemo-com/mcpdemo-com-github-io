Web Search — Live Demo

Ask anything. Results come from the live web right now — not Claude's training data.

Try a starter prompt
Searching the live web via MCP…
Live result MCP Tool: websearch_web_search

When you hit Search, this page posts your query to a Cloudflare Worker running at api.mcpdemo.com/proxy/web-search. No API key needed on your end — the Worker handles authentication.

The Worker calls websearch_web_search directly on the mcpcio.com MCP server at https://api.mcpcio.com/mcp using the JSON-RPC 2.0 protocol. This is a direct tool call — there is no Claude or LLM in the search itself. The MCP server runs a live web search and returns a structured answer with source attribution.

The Worker strips noise and returns clean JSON to this page: the synthesized answer and the sources it came from. What you see is what the web actually says right now.
