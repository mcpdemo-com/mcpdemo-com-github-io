What's happening under the hood?

When you search, the demo calls a Cloudflare Worker at api.mcpdemo.com. The Worker holds API credentials securely — they're never sent to your browser.

The Worker uses the mcpcio.com MCP server to call productsearch_search_products — a live tool that queries millions of real Shopify products in real time. Checkout tools are deliberately excluded so results stay on this page.

Responses are cached for 7 days and rate-limited to keep the demo free and fair for everyone. Want the full technical walkthrough? Read the info page →
