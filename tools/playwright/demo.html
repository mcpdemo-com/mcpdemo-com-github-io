What's happening under the hood?

When you run a task, the demo calls a Cloudflare Worker at api.mcpdemo.com. The Worker holds the Anthropic API key securely — it's never sent to your browser.

The Worker calls the Anthropic Messages API with a system prompt that instructs Claude to respond as if it has live Playwright MCP access. Claude describes the exact tool calls it would make — browser_navigate, browser_snapshot, browser_click — and what the accessibility tree would return.

A live Playwright MCP setup runs @playwright/mcp@latest locally via npx and connects to Claude Desktop, VS Code, Cursor, or Windsurf. The browser launches on your machine, reads real pages, and Claude interacts with actual elements by reference ID. Read the full setup guide →
