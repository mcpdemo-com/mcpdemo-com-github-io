When you search, this page posts your query to a Cloudflare Worker at api.mcpdemo.com/proxy/web-search. The Worker makes two parallel MCP tool calls: websearch_query for the synthesized text answer and websearch_video_search for related videos — both direct calls to the mcpcio.com MCP server, no LLM in the loop.

Inline citations from the answer (e.g. [Wikipedia]) are extracted and rendered as numbered superscripts with a footnote list below. Video thumbnails link directly to the source. What you see is what the web actually says right now.
