How does this work?

When you send a query, this page calls the Anthropic Messages API directly from your browser, passing the mcpcio MCP server as a connected tool source.

Claude receives your message and automatically decides to call productsearch_search_products — a tool exposed by the MCP server. The server queries Shopify's product catalogue in real time and returns structured results. Claude then formats those results into a readable reply.

Your API key stays in your browser tab only — it is sent directly to api.anthropic.com and is never logged or stored by MCP Demo.
