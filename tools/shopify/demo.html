What's happening under the hood?

This demo connects Claude directly to Shopify's product catalogue via Model Context Protocol (MCP). When you send a message, Claude calls productsearch_search_products — a live MCP tool that queries millions of real Shopify products in real time.

The MCP server is hosted by mcpcio.com, which handles authentication, rate limiting, and result formatting on your behalf. Your searches are private — no tracking, no data selling.

Want to build this into your own app? See the Shopify MCP info page for the full code example using the mcp-client-2025-04-04 beta header.
