What's happening under the hood?

When you search, this demo calls a Cloudflare Worker at api.mcpdemo.com. The Worker holds API credentials securely — they're never sent to your browser.

The Worker calls the Anthropic Messages API with GitHub's official remote MCP server (api.githubcopilot.com/mcp/) configured as a live tool. Claude makes real GitHub API calls through the Model Context Protocol and returns the results in plain English.

Access is read-only — the Worker uses a scoped token with no write permissions. Claude can search and read, but nothing on GitHub can be created, modified, or deleted through this demo. Read the full technical walkthrough →
